Раздел 3 | Статистика

Команда для вызова "Статистика": /curinfo или "Статистика"

Статистика - раздел с данными о проекте.

Чаты - количество чатов, где бот побывал (и получил подтверждение)
Пользователи - количество пользователей, причём отдавшие согласие на использование данных
Запросов - количество запросов на погоду через API
Всего локаций - количество локаций, которыми интересовались другие пользователи
