# Telegram Weather Bot
## Краткое описание:
Проект для ДиФолио (портфолио) | Телеграм бот, который отправляет погоду за сегодня и завтра, имеющий админку, несколько команд, система хранения данных в датабазе, созданный для Uchi.Doma "Код будущего", за 4 дня
## Что содержит данный проект?
Этот проект имеет телграм бота, считывающий погоду. Кроме бота есть два отдельных самостоятельно-созданных API для обращения к погодам.
### 1. program.py - Основной бот
Бот в принципе построен на синхронной библиотеке Telebot
**ТОКЕН БЕРЁТСЯ ИЗ ФАЙЛА VCWAPIKEY.txt**
Возможности:

- Узнать погоду по геолокации, имени места, позиции или американского почтового индекса и рассказывать про текущий и завтрашний день, текущее время
- Хранить векрифицированных пользователей и их чаты, а также заданные ими места
- Обработчик большинства ошибок, которые могут произойти при получении погоды
- Админ панель
- Читать и записывать датабазу при открытии\закрытии проги (а также сохранять дистанционно через админку)
- И по мелочи (Раздел Помощи, рассылка сообщений, выключение бота и т.д.)

### 2. MOD_VisualCrossingWeather.py - API Visual Crossing Weather
Этот файл содержит простенькую работу Visual Crossing Weather API через GET запросы, а также немалое кол-во обработок ошибок.

Возможности:

- Поиск места по имени и по позиции
- Много ошибок, по которым легко обрабатывать результаты

### 3. MOD_OpenMeteo.py - API OpenMeteo
Этот файл уже содержит работу OpenMeteo API через GET запросы, но там не настолько много контента, как во 2 файле.

Возможности:

- Поиск места по позиции
- Гибкое управление
